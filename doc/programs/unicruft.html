NAME

unicruft - UTF-8 approximator

SYNOPSIS

unicruft [OPTIONS] FILE(s)

 Arguments:
    FILE(s)  Input text file(s)
 Options
    -h        --help                  Print help and exit.
    -V        --version               Print version and exit.
    -cFILE    --rcfile=FILE           Read an alternate configuration file.
    -mSTRING  --mode=STRING           Conversion mode (ua|ul|ud|udpp|lu)
    -a        --utf8-to-ascii         Convert UTF-8 input to 7-bit ASCII (default)
    -l        --utf8-to-latin1        Convert UTF-8 input to 8-bit Latin-1
    -d        --utf8-to-latin1-de     Convert UTF-8 input to de_DE.Latin-1-'safe' UTF-8
    -p        --utf8-to-latin1-de-pp  Preprocess UTF-8 input to UTF-8 output (debug)
    -u        --latin1-to-utf8        Convert Latin-1 input to UTF-8 output
    -oFILE    --output=FILE           Output file (default=stdout).
